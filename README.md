# NCSA SQC Part 1 – Security Assessment

## Overview

This repository documents my completion of Part 1 of the NCSA Skills Qualification Check (SQC). The objective of this lab was to perform network enumeration, wireless reconnaissance, vulnerability assessment, exploitation, and password cracking using industry-standard cybersecurity tools.

---

## Objectives

- Perform an Nmap service scan
- Identify wireless access points
- Conduct a vulnerability assessment with OpenVAS
- Perform OSINT using Google Dorks
- Exploit a vulnerable FTP service using Metasploit
- Crack a password hash using Hashcat or John the Ripper

---

## Tools Used

- Kali Linux
- Nmap
- Aircrack-ng
- OpenVAS
- Metasploit Framework
- Hashcat
- John the Ripper
